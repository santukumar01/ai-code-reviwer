## ğŸ“¦ Installation

1.  Clone the repository:

    ```bash
    git clone https://github.com/USERNAME/REPO.git
    ```

2.  Navigate to the project directory:

    ```bash
    cd REPO
    ```

3.  Install backend dependencies:

    ```bash
    cd Backend
    ```
